------------
Installation
------------

1. Copy the JavaMail API (mail-1.4.jar) and Activation API (activation-1.1.jar) JARs to
   the container library directory (in Tomcat 6.0: /lib)
2. Setup STMP server such as JES (http://www.ericdaugherty.com/java/mailserver/)
   NB: If using JES, set the dns.server environment variable when starting the server.
